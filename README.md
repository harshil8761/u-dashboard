# 🚀 E-Commerce Dashboard (React.js)

A fully functional E-Commerce Dashboard built using React.js with authentication, protected routing, session management, profile editing, and cart functionality.

---

## 📌 Features

### 🔐 Authentication System
- User Registration
- User Login
- Protected Routes
- 5-Minute Session Expiry
- Automatic Logout
- Live Session Countdown Timer

### 👤 User Profile
- View User Details
- Edit Name, Email, Password
- Real-time UI Update
- LocalStorage Persistence

### 🛒 Cart System
- Add to Cart
- Increase / Decrease Quantity
- Remove Items
- Dynamic Total Calculation
- Toast Notifications

### 🎨 UI & Layout
- Responsive Sidebar Layout
- Fixed Navbar
- Clean Dashboard UI
- Reusable Components
- Tailwind CSS Styling

---

## 🛠️ Tech Stack

- React.js
- React Router DOM
- Context API
- Tailwind CSS
- React Toastify
- LocalStorage (Session & Data Handling)
